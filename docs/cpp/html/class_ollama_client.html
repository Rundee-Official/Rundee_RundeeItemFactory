Detailed Description

Static class for communicating with Ollama LLM server.

Provides methods to send prompts to Ollama and receive JSON responses. Uses HTTP API (WinHTTP on Windows) for communication.

Simple helper function to call Ollama HTTP API.

Makes a single HTTP request to Ollama with the given model and prompt. Uses the provided OllamaSettings for connection configuration.

Parameters
modelNameName of the LLM model (e.g., "llama3", "mistral")
promptPrompt text to send to the LLM
settingsOllama connection settings (host, port, timeouts)
Returns
JSON response as a string, or empty string on error
Note
This function does not retry on failure
Uses WinHTTP on Windows for HTTP communication

Run LLM call with automatic retry logic.

Attempts to call the LLM with automatic retry on failure. Uses settings from AppConfig (loaded from config/rundee_config.json).

Parameters
modelNameName of the LLM model (e.g., "llama3", "mistral")
promptPrompt text to send to the LLM
maxRetriesMaximum number of retry attempts (default: 3)
timeoutSecondsTimeout per attempt in seconds (default: 120, 0 = no timeout)
Returns
Response string, or empty string if all attempts failed
Note
Automatically loads OllamaSettings from AppConfig
Retries with exponential backoff on failure
Logs retry attempts to std::cerr
See also
RunSimple for single-call version
