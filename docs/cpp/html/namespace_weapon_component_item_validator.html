void WeaponComponentItemValidator::Validate ( ItemWeaponComponentData & item)

Validate and balance weapon component item data.

Performs comprehensive validation and balancing on a weapon component item:

  • Normalizes ID prefix to "WeaponComponent_"
  • Validates component type
  • Clamps stat modifiers to valid ranges
  • Generates default description if empty
Parameters
[in,out]itemWeapon component item to validate and balance (modified in-place)
Note
This function modifies the item parameter directly
