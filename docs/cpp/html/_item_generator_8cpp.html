Detailed Description

Implementation of high-level item generation using LLM (Ollama).

Author
Haneul Lee (Rundee)
Date
2025-11-15
Copyright
Copyright (c) 2025 Haneul Lee. All rights reserved.

This file implements the core item generation logic for RundeeItemFactory. It supports parallel batch processing, quality checking, duplicate detection, and registry-based ID tracking across multiple item types.

Key Features:

  • Parallel batch processing for efficient LLM interaction
  • Thread-safe file I/O using per-file mutexes
  • Quality checking and validation for generated items
  • Registry-based duplicate ID prevention
  • Support for all item types: Food, Drink, Material, Weapon, WeaponComponent, Ammo, Armor, Clothing
